# Machine Learning Projects

A collection of machine learning and data analysis notebooks, completed as part of my data science minor course at the American University of Afghanistan (AUAF), in spring of 2022, built with Python and Jupyter, covering regression, classification, exploratory data analysis, and data preprocessing.

## Projects

| # | File | Title | Description |
|---|------|-------|-------------|
| Project 1 | `Project_1.ipynb` | Final Project — Abalone Shell Analysis | Linear & Logistic Regression on abalone dataset; EDA, label encoding, model evaluation |
| Project 2 | `Project_2.ipynb` | Midterm — Car Workshop Orders | Data cleaning, merging 3 Excel sheets, spelling correction, duplicate removal |
| Project 3 | `Project_3.ipynb` | Quiz — Titanic Passenger Analysis | EDA, filtering, apply functions, age discretization, custom title extraction |
| Practice | `Practice.ipynb` | Working Directory — ML Practice | Pandas operations, datetime, groupby, Linear Regression, gradient descent, one-hot encoding |

## Datasets

| Dataset | Used In | Description |
|---------|---------|-------------|
| `abalone.csv` | Project 1 | Physical measurements of abalone shells |
| `carWorkshop-1.xlsx` | Project 2 | Car workshop orders across 3 sheets (orders, plates, services) |
| `test.csv` | Project 3 | Titanic passenger data |
| `Purchase_Data-2.csv` | Practice | Purchase and invoice records |
| `Grades.csv` | Practice | Student grades and participation |
| `Titanic.csv` | Practice | Titanic passenger data |

| `mcdonalds_menu.csv` | Practice | McDonald's menu nutritional data |

| `winequality-red.csv` | Practice | Red wine physicochemical properties and quality ratings |

## Tech Stack

- Python 3
- Jupyter Notebook
- Pandas · NumPy · Matplotlib · scikit-learn · openpyxl

## Topics Covered

- Exploratory data analysis (EDA)
- Data cleaning and preprocessing
- Merging and reshaping DataFrames
- Datetime operations
- Apply functions and custom discretization
- Linear and Logistic Regression
- Multiple linear regression
- Gradient descent implementation
- One-hot encoding and label encoding
- Model evaluation (RMSE, MSE, accuracy)
- Train/test split and cross-validation
