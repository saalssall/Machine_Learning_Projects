# Machine Learning Projects

A collection of machine learning and data analysis assignments built with Python and Jupyter Notebooks, covering regression, classification, and exploratory data analysis.

## Projects

| # | Title | Dataset | Techniques |
|---|-------|---------|------------|
| Final Project | Abalone Shell Analysis | abalone.csv | Linear Regression, Logistic Regression, EDA |
| Assignment 03 | Car Workshop Orders | carWorkshop-1.xlsx | Data Cleaning, Merging, Pandas |
| Quiz 01 | Titanic Passenger Analysis | test.csv | EDA, Filtering, Apply Functions |

## Tech Stack

- Python 3
- Jupyter Notebook
- Pandas · NumPy · Matplotlib · scikit-learn

## Getting Started

```bash
git clone https://github.com/saalssall/Machine_Learning_Projects.git
cd Machine_Learning_Projects
pip install pandas numpy matplotlib scikit-learn jupyter openpyxl
jupyter notebook
```

## Topics Covered

- Data cleaning and preprocessing
- Merging and reshaping DataFrames
- Exploratory data analysis (EDA)
- Linear and Logistic Regression
- Model evaluation (RMSE, MSE, accuracy)
- Label encoding and feature engineering
- Apply functions and custom discretization
